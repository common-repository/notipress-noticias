Plugin para la obtenci&oacute;n de contenidos de la agencia de noticias NotiPress.

== Description ==

El complemento NotiPress Noticias es la manera m&aacute;s f&aacute;cil de tener acceso a noticias de negocios, tecnolog&iacute;a, econom&iacute;a, internacionales y estilo de vida, entre otros contenidos. La agencia de noticias NotiPress con sede en Ciudad de M&eacute;xico pone a disposici&oacute;n de los medios de comunicaci&oacute;n el plugin para descargar notas, directo a WordPress.

Para activar el plugin, el medio de comunicaci&oacute;n deber&aacute; contar con una clave de licencia. Podr&aacute; [solicitar una licencia de prueba](https://ntp.re/notifree) sin costo.


== Installation ==

Pasos para instalar el plugin.

1. Subir el plugin a la carpeta `/wp-content/plugins/notipress-noticias`, o iniciar la instalaci&oacute;n desde el directorio de WordPress directamente.
2. Activar el plugin en WordPress.
3. Ir a la configuraci&oacute;n del plugin NotiPress Noticias para introducir la clave de la licencia.
4. Ya puede comenzar a usar el plugin desde el menu NotiPress

== Frequently Asked Questions ==

### ¿Qu&eacute; hace exactamente este plugin?

Facilita la obtenci&oacute;n de contenidos (noticias) de la plataforma de NotiPress. Con solo unos clics, es posible tener una nueva entrada en borrador.

### ¿A qui&eacute;n est&aacute; dirigido el plugin? 

Especialmente a editores de medios informativos y profesionales de la comunicaci&oacute;n cuya funci&oacute;n sea curar contenidos, editar y publicar.
